This REST API provides the basic functionalities to create a clone of a social network. 
It allows users to create accounts, post content, interact with other users, and more.

## Features
 - nodejs 18 latest
 - typescript 4.9.5
 - express 4.18.2
 - firebase 9.20.0

# How to install
  npm install
  npm run dev

# compile ts to js
  tsc --watch
  
# run server
  npm run start
